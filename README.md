# Nand2Tetris | Personal Implementation

This repository contains my personal implementation of the projects from
The Elements of Computing Systems and the Nand2Tetris course.

The course builds a full computing stack step by step:

* logic gates
* CPU
* assembler
* virtual machine
* compiler
* operating system

The goal is to understand how a computer works from the lowest level up.

## Why this repository is public

The course authors recommend not publishing solutions.

I understand the reason for that. The goal of the course is learning, and people should try to solve the projects themselves.

I still decided to keep this repository public for a simple reason: this is a personal hobby project. Today anyone can generate solutions with AI in seconds. My code is not meant to be the best or the cleanest reference implementation. It is just the result of me working through the material step by step.

If you are taking the course, you should try solving the projects yourself instead of looking at solutions.

## How I work on this project

This project is something I do for fun.

Sometimes late at night I open my computer, start **vim**, and write simple code by hand. No frameworks, no complicated setup, just a text editor and a compiler.

I try to rely on my own understanding as much as possible and use minimal AI help. The goal is not speed. The goal is to understand the details.

Writing small pieces of code slowly and thinking about how things work is relaxing for me.

## What this project represents

This repository is not meant to impress recruiters and it is not meant to be a polished production project.

It is closer to a notebook.

Working through these projects is like a long meditation.
